# React + Vite blueprint App
For quick project setup including paths, routes, pages etc.
Includes:
- Tailwind CSS
- TypeScript
- pnpm package manager

## Start
You should use [pnpm](https://pnpm.io/) package manager :

```
cd frontend
pnpm install
pnpm run dev 
```

### Important Versions
- "node.js": v22.14.0
- "vite": "^6.2.0"
- "typescript": "~5.7.2",
- "@tailwindcss/vite": "^4.1.3",
- "react": "^19.0.0",
